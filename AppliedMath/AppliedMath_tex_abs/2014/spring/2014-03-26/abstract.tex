\newcommand{\talktitle}{Tail Mean Estimation \break is More Efficient than Tail Median: \break Evidence from the Exponential Power Distribution}

\newcommand{\talkspeaker}{ \textbf{\sc Roger Barnard}\\ \textit{Texas Tech University}}

\newcommand{\talkdate}{\textbf{Wednesday, March 26, 2014}}

\newcommand{\timelocation}{\textbf{Room: MATH 111.  Time: 4:00pm.}}

\newcommand{\talkabstract}{
In this talk we will show how the properties of the Incomplete Gamma function
can be used to compare the asymptotic efficiency of the Tail Mean Estimation
to the Tail Median Estimation for the Exponential Power Distribution.
